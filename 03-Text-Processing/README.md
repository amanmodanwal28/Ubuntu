- awk
- cat
- column
- cut
- frep
- head
- less
- more
- paste
- sed
- sort
- tac
- tee
- tail
- tr
- uniq
- wc
- zcat

## Objectives

- Search log files
- Filter data
- Process text efficiently
- Analyze command output
